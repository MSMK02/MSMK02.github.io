Hello World~

Hello World一般是程序员学习编程的第一个程序，典型如K&R的the C programming language，一开始讲述C语言编程的时候，
就是用这个程序来的头，从此引领了程序语言类书籍的潮流，逢讲编程语言，必先以此程序来头，帮助程序员打开编程世界的大门。

当然还有一部同名番剧 Hello World 这部动漫，个人十分推荐！！

剧情简介

2027年住在日本京都市的主人公坚书直实，被从十年后2037年来的未来的自己告知，自己的所住的世界只是真实世界的数据，
而未来的自己是来帮他找女朋友的，随后告诉了自己，在与她刚交往没多久的一个烟火大会中，她就遇到事故身亡了，
希望过去的自己能回避掉死亡的悲剧，虽然知道无法改变未来，只是希望能看见她的笑容而已。但随着时间过去……一切似乎并非如此。
“世界在最后一秒，被完全颠覆”
